# Site Web Agence de Voyage

Ce projet est un site web statique pour une agence de voyage. Il utilise HTML et CSS pour créer un design simple et responsive.

## Fonctionnalités

- **Responsive**
- **Navigation fluide** 
- **Formulaire de contact** 

## Technologies

- **HTML5**
- **CSS3**
